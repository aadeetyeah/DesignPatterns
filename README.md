# DesignPatterns

## **A. Creational Design Patterns**

### 1. **Simple Factory**

When to use Simple Factory Design Pattern?

=> When we create similar objects but the creation process is slightly different.
    e.g.: DeliveryOrder, TakeOutOrder, DineInOrder, etc.
